LearnMyWay
An AI-powered web app that teaches school topics through your personal interests.
What it does
Pick an interest — Anime, Gaming, Football, Fashion, Beauty or TikTok — type in a school topic like Photosynthesis or Gravity, and get a personalized lesson generated just for you in a storytelling style.
How to use

Select your interest
Type your school topic
Click Generate
Read your personalized lesson

Tech Stack

React + Vite
Groq API (llama-3.3-70b-versatile)
Deployed on Vercel

Purpose
Portfolio project built in 7 days to demonstrate AI integration and product thinking.
